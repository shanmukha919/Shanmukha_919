## ğŸ¨ Customization

### Update Personal Information

1. **Profile Images**: Replace placeholder images in `src/assets/images/`
2. **Resume**: Add your resume PDF to `public/resume.pdf`
3. **Personal Details**: Update information in the respective components:
   - `src/components/Hero.jsx` - Name, role, description
   - `src/components/About.jsx` - Bio, experience, education
   - `src/components/Skills.jsx` - Your skills and proficiency
   - `src/components/Projects.jsx` - Your projects
   - `src/components/Contact.jsx` - Contact information

### Theme Colors

Edit CSS variables in `src/styles/global.css`:

```css
:root {
  --primary-color: #00abf0;
  --bg-color: #081b29;
  --second-bg-color: #112e42;
  --text-color: #ededed;
}
```
